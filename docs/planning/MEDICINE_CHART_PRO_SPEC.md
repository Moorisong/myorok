# 유료버전 – 약/영양제 복용 차트 UI/UX 명세

## 목적
기간(15일 / 1개월 / 3개월 / 전체)이 변경되더라도 약/영양제 복용 이력을 **오해 없이, 과밀하지 않게, 의료적으로 의미 있게** 제공한다.

---

## 기간 선택 옵션

- 15일
- 1개월
- 3개월
- 전체

기간 변경은 **차트 표현 밀도(density)만 변경**하며, 의미 구조는 유지한다.

---

## 공통 구조

- Y축: 약/영양제 목록
- X축: 선택된 기간의 시간 흐름
- 약/영양제별 **단일 행(row)** 유지
- 표현 방식은 복용 패턴에 따라 자동 전환

---

## 기간별 UI/UX 전략

### 1️⃣ 15일 / 1개월 (단기 구간)

#### 표현 방식
- 날짜 단위 표현 유지
- 연속 복용: 연속 막대(bar)
- 비연속 복용: 날짜별 점(dot)

#### 목적
- 정확한 날짜 확인
- 복용 누락 파악
- 최근 순응도 판단

---

### 2️⃣ 3개월 (중기 구간)

#### 문제점
- 날짜 단위 점 표현 시 과밀 발생
- 시각적 노이즈 증가

#### 표현 방식
- **주 단위 집계(week-based aggregation)**

| 패턴 | 표시 |
|----|----|
| 주 5일 이상 복용 | 굵은 막대 |
| 주 2~4일 복용 | 얇은 막대 |
| 주 1일 이하 | 점 |
| 미복용 | 공백 |

#### 보조 정보
- 툴팁: `3월 2주차 – 4/7일 복용`

---

### 3️⃣ 전체 기간 (장기 히스토리)

#### 기본 원칙
- 날짜/주 단위 표현 ❌
- **패턴 요약 중심**

#### 표시 방식 (요약 카드형)

- 시작일 ~ 마지막 복용일
- 전체 복용 기간
- 총 복용일 수
- 평균 복용 빈도

예:
```text
A 유산균
복용 기간: 2024.01 ~ 2025.03
총 142일 복용
평균: 주 4.2회
```

#### 차트 표현
- **밀도 히트바(density bar)** 또는
- 연속/비연속 구간 분절된 막대

---

## 약/영양제 전용 UX 규칙

### 자동 패턴 분류
- 시스템이 연속 / 비연속 / 간헐 / 중단 상태를 자동 판단
- 사용자는 판단 결과를 수정할 수 없음 (의료용 안정성)

### 중단 표시
- 일정 기간(예: 30일) 미복용 시
  - 상태: `중단됨`
  - 시각적으로 흐리게 표시

---

## UX 일관성 원칙

- 기간이 길어질수록 → **정보는 요약**
- 기간이 짧을수록 → **정보는 정확**
- 어떤 기간에서도
  - “연속 복용 오해” 발생 금지
  - 복용 빈도는 반드시 확인 가능

---

## 요약 정책

- 15일 / 1개월: 날짜 기반 정확 표현
- 3개월: 주 단위 밀도 표현
- 전체: 패턴 요약 중심

---

## 한 줄 결론

기간이 길어질수록  
**약/영양제 복용 차트는 ‘날짜 기록’에서 ‘복용 패턴 요약’으로 진화한다.**
