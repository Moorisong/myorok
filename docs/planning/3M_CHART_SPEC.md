# 📊 3개월 차트 개선 명세 (3M Range)

## 개요
15일 / 1개월 차트는 "일 단위 이벤트"를 정확히 보여주는 것이 목적이다.  
그러나 3개월(약 90일)은 동일한 표현 방식을 사용할 경우 가독성과 의료적 해석력이 급격히 저하된다.

따라서 **3개월 차트는 일 단위가 아닌 집계 기반 요약 차트**로 표현한다.

---

## 적용 범위
- Charts 탭 → Summary
- 기간 선택: **3개월(3M)**

---

## 설계 원칙

- 일 단위 이벤트 표현 ❌
- 밀도 축소 ⭕
- 숫자 중심 ⭕
- "어느 기간이 문제였는지" 즉시 파악 ⭕

---

## 3개월 차트 표현 전략

### 핵심 전략
> **3개월 = 주 단위(Weekly) 집계 차트**

- 약 12주 단위로 데이터를 요약
- 차트 개수는 유지하되, 데이터 포인트 수 감소

---

## 1. 증상 차트 (배변 / 설사 / 구토)

### 차트 타입
- **Bar Chart (주 단위 집계)**

### 데이터 집계 기준
| 항목 | 기준 |
|---|---|
| 집계 단위 | 주(7일) |
| 값 | 해당 주의 총 발생 횟수 |
| 단위 | 회 |

### 표현 규칙

| 항목 | 규칙 |
|---|---|
| X축 | Week 1 ~ Week N |
| Y축 | 주간 합계 (정수) |
| 표시 | 세로 막대 |
| 수치 | 막대 상단에 `N회` 고정 표시 |
| 0인 주 | 막대 없음 (빈 공간) |
| 스크롤 | 데이터 로드 시 최신 주(우측 끝)로 자동 이동 |

---

## 2. 강수 / 수액 차트 (3개월)

### 차트 타입
- **Bar Chart (주 단위 합산)**

### 표현 규칙

| 항목 | 규칙 |
|---|---|
| 집계 단위 | 주 |
| 막대 | 주 단위 합산 |
| 타입 구분 | 💧 강수 / 💉 수액 |
| 수치 | 막대 상단에 `N ml` 고정 표시 |
| 0인 주 | 막대 없음 |

---

## 3. 기간별 차트 책임 정리

| 기간 | 표현 목적 | 방식 |
|---|---|---|
| 15일 | 며칠에 몇 번 | 일 단위 Dot |
| 1개월 | 최근 패턴 확인 | 일 단위 Dot |
| **3개월** | 문제 구간 파악 | **주 단위 Bar** |
| 전체 | 전반적 관리 상태 | 통계 카드 |

---

## 한 줄 요약
> 3개월 차트는 "상세 기록"이 아니라 "문제가 있었던 기간을 찾기 위한 요약 지도"다.
