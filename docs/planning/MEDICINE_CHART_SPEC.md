# 병원용 요약 차트 – 최근 15일 약/영양제 복용 표시 명세

## 목적
최근 15일간의 약/영양제 복용 이력을 **의사가 오해 없이 3초 내 파악**할 수 있도록 시각화한다.  
연속 복용과 비연속(띄엄띄엄) 복용을 명확히 구분한다.

---

## 기본 구조

- 영역명: 최근 15일 약/영양제 복용
- 차트 형태: 약/영양제별 가로 타임라인
- X축: 최근 15일 날짜 (D-14 ~ D)
- Y축: 약/영양제 이름
- 차트 구조는 유지하며, **표현 방식만 분기**한다.

---

## 데이터 전제

- 복용 데이터는 날짜 단위로 존재한다.
- 하루라도 복용 기록이 없으면 해당 날짜는 공백으로 처리한다.

예시 데이터:
- A 유산균: 1/1, 1/3, 1/6, 1/10 복용
- B 혈압약: 1/1 ~ 1/10 연속 복용

---

## 표현 규칙

### 1. 연속 복용 (Continuous Intake)

#### 조건
- 복용 날짜가 하루 단위로 끊김 없이 연속된 경우

#### 표시 방식
- 하나의 **연속 막대(bar)** 로 표시
- 시작일 ~ 종료일까지 연결된 막대 사용

#### 툴팁 / 보조 정보
- 예: `연속 10일 복용`
- 필요 시 시작일/종료일 표시 가능

#### 금지 사항
- 비연속 복용임에도 기간 막대로 묶어 표시하는 것 금지

---

### 2. 비연속 복용 (Intermittent Intake)

#### 조건
- 복용 날짜 사이에 1일 이상 공백이 존재하는 경우

#### 표시 방식
- **복용한 날짜에만 점(dot) 또는 짧은 막대 표시**
- 날짜 간 연결 금지

#### 툴팁 / 보조 정보
- 예: `최근 15일 중 4회 복용`
- 각 점 hover 시 해당 날짜 표시

#### 금지 사항
- `1/1 ~ 1/10`과 같은 기간 텍스트 표시 금지
- 연속 복용처럼 보이게 하는 시각적 연결 금지

---

## UX 원칙 (병원용)

- 정확성 > 미관
- 연속 복용으로 오해될 수 있는 표현 금지
- 복용 누락 여부가 한눈에 보여야 함
- 의학적 판단(순응도, 복약 이탈) 보조 목적

---

## 요약 규칙

- 연속 복용 = 기간 막대
- 비연속 복용 = 날짜별 점 표시
- 같은 차트, 다른 표현 방식
- 텍스트 기간 요약은 연속 복용에만 허용

---

## 한 줄 결론

최근 15일 복용 차트는 유지하되,  
**연속은 막대 / 비연속은 점으로 표현하여 의료적 오해를 방지한다.**
