# 무료 / 유료 서비스 구분

> 광고 없음

---

## 기능 비교표

| 기능 구분 | 무료 사용자 | 유료 사용자 |
|----------|------------|------------|
| 기본 컨디션 기록 (배변·배뇨·구토 등) | ✅ 가능 | ✅ 가능 |
| 사료 기호성 기록 | ✅ 가능 | ✅ 가능 |
| 약 / 영양제 기록 | ✅ 가능 | ✅ 가능 |
| 병원 방문 기록 | ✅ 가능 | ✅ 가능 |
| 커스텀 수치 생성 | 제한적 (최대 3개) | 무제한 |
| 커스텀 수치 기록 | ✅ 가능 | ✅ 가능 |
| 기본 차트 (배변·구토 추이) | 최근 15일 | 전체 기간 |
| 커스텀 수치 차트 | 최근 15일 | 전체 기간 |
| 데이터 보관 기간 | 최근 15일 | 무제한 |
| 과거 데이터 조회 | 흐리게 표시 (읽기만) | 정상 표시 |
| 병원용 차트 화면 | ❌ | ✅ |


---

## 핵심 원칙

- **광고 없음**: 무료/유료 모두 광고 없음
- **데이터는 항상 저장**: 삭제하지 않음
- **무료는 "덜 보임"**: 접근 제한만 있고, 데이터 자체는 존재
